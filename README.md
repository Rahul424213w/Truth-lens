# TruthLens - AI-Powered News Credibility Platform

TruthLens is a project that uses Google's Gemini AI to analyze the credibility of news content through text, image, and article analysis.

## Features

- **Text Analysis**: Analyze news text for credibility, red flags, and potential misinformation
- **Image Analysis**: Detect manipulated or AI-generated images
- **Article Analysis**: Comprehensive analysis of news articles including source credibility
- **Source Comparison**: Compare claims from multiple sources for consistency
